To install the base system, run this in a terminal:

wget -q -O - http://apt.genome.wustl.edu/ubuntu/files/install.sh | sudo bash

For a list of components available, run “genome install” with no parameters.

genome install

Add components with the “genome install” command, i.e.:

genome install joinx1.1

Behind the Scenes:

Debian (including Ubuntu) GNU/Linux systems

Genome Modeling System uses the native apt package manager, and links to an apt repository served from apt.genome.wustl.edu for systematic updates.

repository linkage:


sudo apt-add-respository "deb http://apt.genome.wustl.edu lucid-genome main"
wget http://apt.genome.wustl.edu/ubuntu/files/genome-center.asc | sudo apt-key add
sudo apt-get update

install joinx:


sudo apt-get install joinx1.1

Other Platforms/source code:

The full source code and build instructions are available from the github project page.

Help:

To get help on joinx or a particular subcommand, use the -h option, e.g.:


joinx -h
joinx intersect -h

Manuals:

To see the manual page:


man joinx1.1
