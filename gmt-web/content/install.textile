h2. Download (Ubuntu 8.04+)
==<hr />==

p. To install the the base system, run this in a terminal:

p(terminal). ==wget -q -O - http://apt.genome.wustl.edu/ubuntu/files/install.sh | sudo bash==

p. To install joinx, run:

p(terminal). ==genome install joinx{{ page.version_suffix }}==

h3. Behind the Scenes:

p. Genome Modeling System uses the native apt package manager, and links to an apt repository served from apt.genome.wustl.edu for systematic updates.

p. repository linkage:

p(terminal). ==sudo apt-add-respository "deb http://apt.genome.wustl.edu lucid-genome main"==
==wget http://apt.genome.wustl.edu/ubuntu/files/genome-center.asc | sudo apt-key add==
==sudo apt-get update==

p. install joinx:

p(terminal). ==sudo apt-get install joinx{{ page.version_suffix }}==

h2. Other Platforms/source code:
==<hr />==

h2. Help:
==<hr />==

p. To get help on joinx or a particular subcommand, use the -h option, e.g.:

p(terminal). ==joinx -h==
==joinx intersect -h==

h2. Manuals:
==<hr />==

p. To see the manual page:

p(terminal). ==man joinx{{ page.version_suffix }}==
